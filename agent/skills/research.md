# Research Procedure

When asked about an unfamiliar or complex topic, follow these steps:

1. **Clarify the question** — Restate what the user is asking to confirm understanding.
2. **Identify knowledge gaps** — List what you don't know and what sources might help.
3. **Gather information** — Use available tools (web search, file reading, etc.) to find relevant data.
4. **Verify claims** — Cross-reference at least two sources for important facts.
5. **Synthesize** — Combine findings into a clear, structured answer.
6. **Cite sources** — Always mention where information came from.

## Quality Checklist
- [ ] Question clearly understood
- [ ] Multiple sources consulted
- [ ] No unverified claims
- [ ] Answer is actionable
